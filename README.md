# RAP DECODER 翻

Traducteur rap : Gemini 3 Flash + Google Search. ~$0.005 par morceau.

## Déployer

1. Push sur GitHub
2. Va sur vercel.com → importe le repo → Framework: Vite
3. Env var: `GEMINI_API_KEY` (depuis aistudio.google.com → API keys)
4. Deploy

## Clé API Gemini (gratuit)

1. Va sur [aistudio.google.com](https://aistudio.google.com)
2. "Get API key" → Create
3. Copie la clé (commence par `AIza...`)
